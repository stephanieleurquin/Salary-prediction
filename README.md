

# Salary Prediction

## Description
Ce projet est un modèle de **prédiction de salaire** développé avec Python dans kaggle.  
L’objectif est de prédire le salaire d’un individu en fonction de ses caractéristiques (par exemple : expérience, poste, compétences, etc.) en utilisant des techniques de machine learning.

---

## Technologies utilisées
- Python 3.x  
- Kaggle  
- Pandas (pour la manipulation de données)  
- NumPy (pour les calculs numériques)  
- Scikit-learn (pour la modélisation ML)  
- Matplotlib / Seaborn (pour la visualisation des données)  

---

## Contenu du projet
- `Salary_Prediction.ipynb` : Notebook principal avec le code pour la préparation des données, la modélisation et l’évaluation du modèle.  
- `data/` : Dossier contenant le dataset utilisé (si applicable).
